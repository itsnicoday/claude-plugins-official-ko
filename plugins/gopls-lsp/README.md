# gopls-lsp

Claude Code를 위한 Go 언어 서버로, 코드 인텔리전스, 리팩터링 및 분석을 제공합니다.

## 지원하는 확장자
`.go`

## 설치

Go 툴체인을 사용하여 gopls를 설치합니다:

```bash
go install golang.org/x/tools/gopls@latest
```

`$GOPATH/bin` (또는 `$HOME/go/bin`)이 PATH에 포함되어 있는지 확인하세요.
