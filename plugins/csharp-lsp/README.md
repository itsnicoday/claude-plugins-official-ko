# csharp-lsp

Claude Code를 위한 C# 언어 서버로, 코드 인텔리전스 및 진단을 제공합니다.

## Supported Extensions
`.cs`

## Installation

### .NET 도구 사용 (권장)
```bash
dotnet tool install --global csharp-ls
```

### Homebrew 사용 (macOS)
```bash
brew install csharp-ls
```

## Requirements
- .NET SDK 6.0 이상
