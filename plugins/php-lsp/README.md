# php-lsp

Claude Code를 위한 PHP 언어 서버(Intelephense)로, 코드 인텔리전스 및 진단 기능을 제공합니다.

## 지원하는 확장자
`.php`

## 설치

npm을 사용하여 Intelephense를 전역으로 설치합니다:

```bash
npm install -g intelephense
```

또는 yarn을 사용하는 방법도 있습니다:

```bash
yarn global add intelephense
```
