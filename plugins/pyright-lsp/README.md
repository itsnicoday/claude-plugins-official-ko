# pyright-lsp

정적 타입 검사 및 코드 인텔리전스를 제공하는 Claude Code용 Python 언어 서버(Pyright)입니다.

## 지원되는 확장자 (Supported Extensions)
`.py`, `.pyi`

## 설치 (Installation)

npm을 통해 전역으로 Pyright를 설치하십시오:

```bash
npm install -g pyright
```

또는 pip를 사용할 수 있습니다:

```bash
pip install pyright
```

또는 pipx를 사용할 수 있습니다 (CLI 도구에 권장됨):

```bash
pipx install pyright
```
