# Frontend Design Plugin

일반적인 AI 스타일의 느낌을 탈피하여 독창적이고 프로덕션 등급의 프론트엔드 인터페이스를 생성합니다.

## What It Does (주요 기능)

Claude는 프론트엔드 작업 시 이 스킬을 자동으로 사용합니다. 다음과 같은 특성을 갖춘 프로덕션 지원 코드를 생성합니다:

- 대담한 미적 디자인 선택
- 독창적인 타이포그래피 및 색상 팔레트
- 임팩트 있는 애니메이션 및 시각적 디테일
- 콘텍스트를 고려한 구현

## Usage (사용법)

```
"Create a dashboard for a music streaming app"
"Build a landing page for an AI security startup"
"Design a settings panel with dark mode"
```

Claude는 명확한 미학적 디자인 방향을 정하고, 세심한 부분까지 디테일을 챙겨 프로덕션 코드를 구현합니다.

## Learn More (더 알아보기)

고품질 프론트엔드 디자인을 위한 프롬프트 가이드라인은 [Frontend Aesthetics Cookbook](https://github.com/anthropics/claude-cookbooks/blob/main/coding/prompting_for_frontend_aesthetics.ipynb)을 참조하십시오.
