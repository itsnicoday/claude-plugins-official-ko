# rust-analyzer-lsp

Claude Code를 위한 Rust 언어 서버로, 코드 인텔리전스 및 분석 기능을 제공합니다.

## 지원하는 확장자
`.rs`

## 설치

### rustup을 통한 설치 (권장)
```bash
rustup component add rust-analyzer
```

### Homebrew를 통한 설치 (macOS)
```bash
brew install rust-analyzer
```

### 패키지 관리자를 통한 설치 (Linux)
```bash
# Ubuntu/Debian
sudo apt install rust-analyzer

# Arch Linux
sudo pacman -S rust-analyzer
```

### 수동 다운로드
[릴리스 페이지](https://github.com/rust-lang/rust-analyzer/releases)에서 미리 빌드된 바이너리를 다운로드합니다.
