# typescript-lsp

Claude Code를 위한 TypeScript/JavaScript 언어 서버로, 정의로 이동(go-to-definition), 참조 찾기(find references), 오류 검사 등의 코드 인텔리전스 기능을 제공합니다.

## 지원하는 확장자
`.ts`, `.tsx`, `.js`, `.jsx`, `.mts`, `.cts`, `.mjs`, `.cjs`

## 설치

npm을 사용하여 TypeScript 언어 서버를 전역으로 설치합니다:

```bash
npm install -g typescript-language-server typescript
```

또는 yarn을 사용하는 방법도 있습니다:

```bash
yarn global add typescript-language-server typescript
```
