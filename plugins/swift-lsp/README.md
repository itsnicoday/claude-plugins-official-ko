# swift-lsp

Claude Code를 위한 Swift 언어 서버(SourceKit-LSP)로, Swift 프로젝트에 대한 코드 인텔리전스를 제공합니다.

## Supported Extensions
`.swift`

## Installation

SourceKit-LSP는 Swift 툴체인에 기본 포함되어 있습니다.

### macOS
App Store에서 Xcode를 설치하거나, 다음 명령어를 통해 Swift를 설치하세요:
```bash
brew install swift
```

### Linux
[swift.org](https://www.swift.org/download/)에서 Swift를 다운로드하여 설치하세요.

설치 후, `sourcekit-lsp`를 PATH에서 사용할 수 있어야 합니다.
